斯坦福机器学习1：线性回归和分类

发表于 2015-07-04   |   分类于 ML   |  

线性回归属于监督学习的一种，解决的问题是根据给定特征值去预测目标值（目标值的集合可为连续，即回归问题；可为离散，即分类问题），回归之意就是通过已有训练集去拟合得到特征值与目标值之间的映射关系，因此问题的输出应是一个函数h。

线性回归的输出应是 y=a0 * x0 + a1 * x1 + a2 * x2（假设有两个特征值, x0 = 1），所以问题就变成了如何求 a0、a1、a2这些系数，这里需要一个表示方法来定量表示所求出的函数是好还是不好，线性回归问题是通过最小二乘定量表示，logistic回归则是通过最大似然率定量表示，得到的都是以所求系数为自变量的函数。这里就引入了梯度下降和牛顿法，二者皆是求解多个参数下如何取得因变量最小值的问题。

线性回归

问题： 系数取何值时， 最小？

解决： 梯度下降或牛顿法迭代

logistic回归
